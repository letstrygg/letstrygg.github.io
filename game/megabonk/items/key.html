Key

+10% chance to open chests for free


Video Explanation

How It Works

Each Key increases your odds, but with diminishing returns — meaning each additional Key adds less benefit than the one before.

The formula is:

Chance to open = n / (n + 10)

Where n is the number of Keys you have.
This means even with infinite Keys, the chance can never exceed 100%.

  • 1 Key → 9.09%
  • 5 Keys → 33.33%
  • 10 Keys → 50%
  • 20 Keys → 66.67%
  • 30 Keys → 75%
