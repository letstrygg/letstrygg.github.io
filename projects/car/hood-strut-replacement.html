How to Replace Hood Struts in a Car

Acura TL (2004–2006) • Applies to most vehicles

Overview

Replacing worn hood struts is a fast, beginner‑friendly DIY repair. If your hood no longer stays open on its own, this fix takes only a few minutes per side and requires minimal tools.

What You’ll Need

Parts

  • Hood struts (pair)

Tools

  • Small flathead screwdriver
  • Small vice grip pliers (Optional)
  • Work gloves (Optional)
Safety: Always support the hood with one hand or use a prop while removing a strut.

Step‑by‑Step Instructions

  1. Secure the hood. Open it fully and keep one hand supporting it. If needed, use vice grips to lock the old strut in place.
  2. Locate the retaining clips. Each end of the strut is held to a ball stud by a small metal clip.
  3. Pry the clip outward. Use a small flathead screwdriver to gently pull the clip away. Do not remove it completely.
  4. Pop the old strut off. Pull the strut straight off the ball stud while supporting the hood.
  5. Install the new strut. Line it up with the ball stud and press firmly until it snaps into place.
  6. Repeat on the other side. Replace the second strut using the same steps.

Final Check

Slowly close and re‑open the hood to make sure both struts are fully seated and the hood stays up on its own.

Repair complete.
